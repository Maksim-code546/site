## Технологии

- **HTML5** – семантическая разметка
- **CSS3 / SCSS** – препроцессор для удобного управления стилями
- **Flexbox / CSS Grid** – построение сеток и раскладок
- **Медиа-запросы** – адаптив под планшеты (768px–1024px) и мобильные устройства (<767px)
- **Google Fonts** – шрифт Montserrat
- **Кастомные чекбоксы/радиокнопки** – стилизация элементов фильтра

## Установка и запуск

1. Склонируйте репозиторий или скачайте архив.
2. Откройте любой из файлов `index.html`, `catalog.html` или `basket.html` в браузере.
3. Для просмотра адаптивного вида используйте инструменты разработчика (Ctrl+Shift+I → режим мобильного устройства).

Никаких дополнительных зависимостей или серверов не требуется – это статический сайт.

Проект выполнен в рамках учебного задания по вёрстке. Все изображения и наименования товаров являются условными.
